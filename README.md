# Shakespeare Scraping
Scraping MIT's [Shakespeare site](http://shakespeare.mit.edu/) to get the text of all of his plays. A sandbox for exploring regex, BeautifulSoup, pandas and matplotlib.

## Built With:
- Python
- BeautifulSoup
- nltk
- textblob
